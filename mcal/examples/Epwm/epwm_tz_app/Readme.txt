Steps to Test epwm_tz_app
=========================

- Connect am263px board to docking station

- Flash & Run the epwm_tz_app in ccs and wait for the text "Wait for sometime and Change the GPIO 122 to Low State from High State."

- Now using a connector wire connect the PIN 72(GPIO122 in am263px-cc) to 3.3v in the docking station board.

  It should proceed with the testing.

-Note: 
 Please verify the PIN number for am263px-lp and am263px-sip boards before testing.
