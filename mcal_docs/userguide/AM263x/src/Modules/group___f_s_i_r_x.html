enum CddFsiRx_StatusType

Enumerator

Value

Description

CDD_FSI_RX_UNINIT

The Rx module has not initialized yet.

CDD_FSI_RX_INIT

The Rx module has initialized.

CDD_FSI_RX_IDLE

The Reception through the Rx channel hasn’t started.

CDD_FSI_RX_BUSY

The Frame reception has been started and is still going on.

CDD_FSI_RX_COMPLETED

The current reception has been completed.

Brief: Current status of the CDD FSI RX HW Unit.
