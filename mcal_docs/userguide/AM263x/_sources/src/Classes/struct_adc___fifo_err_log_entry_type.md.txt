# Adc_FifoErrLogEntryType



ADC log structure per FIFO error entry. 


`#include <Adc_Dbg.h>`

## Structure Members

|  Member Type   | Member name    | Brief |
| -------------- | -------------- | ----- |
|  uint32 | timeStamp |  Timestamp based on 32-kHz Synchronized Timer (COUNTER_32K) at which the FIFO error occurred.  |
|  [Adc_GroupType](../Modules/group___a_d_c.md#typedef-adc_grouptype) | groupId |  Group ID which caused FIFO error.  |
|  [Adc_HWUnitType](../Modules/group___a_d_c.md#typedef-adc_hwunittype) | hwUnitId |  HWUnit associated with the group.  |
