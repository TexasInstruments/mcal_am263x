# Spi_SeqConfigType



SPI Sequence configuration structure. 


`#include <Spi_Cfg.h>`

## Structure Members

|  Member Type   | Member name    | Brief |
| -------------- | -------------- | ----- |
|  Spi_SequenceType | seqId |  Sequence ID.  |
|  uint8 | seqInterruptible |  Sequence interruptible or not (TRUE/FALSE)  |
|  [Spi_SeqEndNotifyType](../Modules/group___s_p_i.md#typedef-spi_seqendnotifytype) | Spi_SequenceEndNotification |  Sequence end notification callback fxn pointer.  |
