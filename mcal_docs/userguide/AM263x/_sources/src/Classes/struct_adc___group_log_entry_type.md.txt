# Adc_GroupLogEntryType



ADC log structure per group entry. 


`#include <Adc_Dbg.h>`

## Structure Members

|  Member Type   | Member name    | Brief |
| -------------- | -------------- | ----- |
|  uint32 | timeStamp |  Timestamp based on 32-kHz Synchronized Timer (COUNTER_32K) at which the group is scheduled.  |
|  [Adc_GroupType](../Modules/group___a_d_c.md#typedef-adc_grouptype) | groupId |  Group ID of the scheduled group.  |
|  [Adc_HWUnitType](../Modules/group___a_d_c.md#typedef-adc_hwunittype) | hwUnitId |  HWUnit associated with the group.  |
|  uint32 | isReScheduled |  This will be set to TRUE is a group is rescheduled when a higher priority task pre-empts this group.  |
