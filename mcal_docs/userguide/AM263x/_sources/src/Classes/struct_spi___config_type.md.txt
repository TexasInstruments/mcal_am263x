# Spi_ConfigType



SPI config structure. 


`#include <Spi_Cfg.h>`

## Structure Members

|  Member Type   | Member name    | Brief |
| -------------- | -------------- | ----- |
|  uint8 | maxChannels |  Maximum number of channels. Should not be more than SPI_MAX_CHANNELS.  |
|  uint8 | maxJobs |  Maximum number of jobs Should not be more than SPI_MAX_JOBS.  |
|  uint8 | maxSeq |  Maximum number of sequences Should not be more than SPI_MAX_SEQ.  |
|  uint8 | maxHwUnit |  Maximum number of HW unit Should not be more than SPI_MAX_HW_UNIT.  |
|  uint8 | maxExtDevCfg |  Maximum number of external device configurations Should not be more than SPI_MAX_EXT_DEV.  |
|  [Spi_ChannelConfigType](../Classes/struct_spi___channel_config_type.md)[SPI_MAX_CHANNELS] | channelCfg |  Channel configurations.  |
|  [Spi_JobConfigType](../Classes/struct_spi___job_config_type.md)[SPI_MAX_JOBS] | jobCfg |  Job configurations.  |
|  [Spi_SeqConfigType](../Classes/struct_spi___seq_config_type.md)[SPI_MAX_SEQ] | seqCfg |  Sequence configurations.  |
|  [Spi_HwUnitConfigType](../Classes/struct_spi___hw_unit_config_type.md)[SPI_MAX_HW_UNIT] | hwUnitCfg |  HW Unit configurations.  |
|  [Spi_ExternalDeviceConfigType](../Classes/struct_spi___external_device_config_type.md)[SPI_MAX_EXT_DEV] | extDevCfg |  External HW device config.  |
|  uint8 | Spi_MaxDlcs |  Maximum number of dlc's.  |
