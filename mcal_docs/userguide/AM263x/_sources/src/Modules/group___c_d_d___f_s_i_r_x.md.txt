### function CddFsiRx_GetStatus

```
CddFsiRx_StatusType CddFsiRx_GetStatus(
    void 
)
```


**Brief**: 
This service gets the status of FsiRx Driver. 

Returns the driver status

Sync/Async - Synchronous

Reentrancy - Reentrant


**Returns**: 

  
* **CDD_FSI_RX_INIT** - Not Initialized 
<br> 
  
* **CDD_FSI_RX_IDLE** - Reception has not started 
<br> 
  
* **CDD_FSI_RX_BUSY** - Reception has been started and still going on 
<br> 
  
* **CDD_FSI_RX_COMPLETED** - Reception completed 
<br> 


**Return**: CddFsiRx_StatusType
