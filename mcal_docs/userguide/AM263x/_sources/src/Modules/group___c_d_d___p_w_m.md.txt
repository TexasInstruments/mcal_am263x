### function Cdd_Pwm_Init

```
void Cdd_Pwm_Init(
    const Cdd_Pwm_ConfigType * ConfigPtr
)
```


**Brief**: 
Function Initializes CDD_PWM unit. 

Cdd_Pwm_Init shall initialize all internals variables and the used PWM structure of the microcontroller according to the parameters specified in ConfigPtr.

Sync/Async - Synchronous

Reentrancy - Non Reentrant


**Parameters**: 

  
* **ConfigPtr** - Pointer to driver configuration. 


**Returns**: 

  
* **None** 
<br> 


**Return**: None
