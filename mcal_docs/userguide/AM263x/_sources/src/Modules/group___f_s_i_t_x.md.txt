### enum Cdd_FsiTx_StatusType
| Enumerator | Value | Description |
| ---------- | ----- | ----------- |
| CDD_FSI_TX_UNINIT | | The Tx module has not initialized yet.   |
| CDD_FSI_TX_INIT | | The Tx module has initialized.   |
| CDD_FSI_TX_IDLE | | The transmission through the Tx channel hasn't started.   |
| CDD_FSI_TX_BUSY | | The Frame transmission has been started and is still going on.   |




**Brief**: 
Current status of the CDD FSI TX HW Unit. 



### enum Cdd_FsiTx_TriggerSourceType
| Enumerator | Value | Description |
| ---------- | ----- | ----------- |
| CDD_FSI_TX_TRIGG_SRC_SW | | Transmission is triggered by a software API call.   |
| CDD_FSI_TX_TRIGG_SRC_HW | | Transmission is triggered by external hardware trigger.   |




**Brief**: 
Type for configuring the trigger source for CDD FSI TX HW. 



### enum Cdd_FsiTx_DataTransmitType
| Enumerator | Value | Description |
| ---------- | ----- | ----------- |
| CDD_FSI_TX_INTERRUPT_MODE | | Transmission using Interrupt Method.   |
| CDD_FSI_TX_POLLING_MODE | | Transmission using Polling Method.   |
| CDD_FSI_TX_DMA_MODE | | Transmission using DMA Method.   |




**Brief**: 
Type for configuring on which edge of the hardware trigger signal the Ping frame should be transmitted. 

Type of CDD FSI_TX Data Transmit method.

### enum Cdd_Fsi_Tx_TransmitLineType
